# Impulse — документация

Impulse — веб-сервис для студентов, которые хотят участвовать в реальных проектах. Приложение построено вокруг двух ключевых сущностей: **Гипотеза** (исследовательская задача от бизнеса) и **Проект** (конкретная команда и реализация, решающая гипотезу).

Текущая версия — клиентский прототип без бэкенда: данные хранятся в памяти сессии и сбрасываются при перезагрузке страницы.

## Роли

В этой документации описана единственная пользовательская роль — **Студент**. Куратор, бизнес-заказчик и администратор на уровне UI не реализованы.

## Ключевые сущности (кратко)

```
Гипотеза (Hypothesis)
  ├── title, shortDescription
  ├── goal, context, tasks, resources, criteria, result  (детали брифа)
  └── projects[ ]  ──► Проект (Project)
                         ├── title, description, status
                         ├── curator, startDate, endDate, tags
                         ├── members[ ]     (команда)
                         ├── vacancies[ ]   (открытые роли)
                         ├── tasks[ ]       (задачи прототипа)
                         └── resources[ ]   (материалы)
```

Одна гипотеза может порождать 0…N проектов. Проект всегда привязан ровно к одной гипотезе через `hypothesisId`.
